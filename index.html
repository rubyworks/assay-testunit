Assay TestUnit

TestUnit Assertions on top of Assay

Assay TestUnit is a compatibility grammar for the Assay test assertions framework.

Assay defines a set of reusable assertion classes in the same way that Ruby defines exceptions.

Assay TestUnit provides a full set of TestUnit compatible assertion methods which translate seemlessly into Assay assertions.

Assay TestUnit is ideal for transitioning to new test frameworks without having to completely rewrite TestUnit-based tests.
